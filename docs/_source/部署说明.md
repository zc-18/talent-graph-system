# 部署说明

> 智岗图谱（TalentGraph AI）　|　题目 XH-202621
> 在线系统：http://101.200.184.201:8200/

系统采用**后端同端口托管前端静态资源**的一体化方案：一个 FastAPI（uvicorn 单 worker）进程即同时提供 REST API 与前端页面，无需额外 nginx，部署轻量、对小内存服务器友好（实测常驻内存约 80MB）。

---

## 一、环境要求

- Python ≥ 3.11（生产服务器使用 3.12）
- Node ≥ 18（仅前端构建需要，建议本地构建）
- MySQL ≥ 8.0（utf8mb4）
- 可访问：DeepSeek API、BGE 向量服务、Tavily、Serper.dev

---

## 二、方式一：systemd + venv（生产推荐，轻量）

### 1. 本地构建前端

```bash
cd frontend
npm install
npm run build        # 产出 frontend/dist
```

### 2. 打包并上传

```bash
# 将后端 app/、.env、requirements.txt 与 frontend/dist（作为 static）打包上传到服务器 /opt/talent-graph/backend/
#   /opt/talent-graph/backend/app
#   /opt/talent-graph/backend/.env
#   /opt/talent-graph/backend/requirements.txt
#   /opt/talent-graph/backend/static      (= frontend/dist)
```

### 3. 创建虚拟环境并安装依赖（仅预编译 wheel，不在服务器编译）

```bash
cd /opt/talent-graph/backend
python3 -m venv .venv
. .venv/bin/activate
pip install --no-cache-dir --only-binary=:all: -r requirements.txt
```

### 4. 配置并启动 systemd 服务

```bash
cp deploy/talent-graph.service /etc/systemd/system/
systemctl daemon-reload
systemctl enable --now talent-graph
systemctl status talent-graph
curl http://127.0.0.1:8200/api/health
```

`talent-graph.service` 关键配置（含内存保护）：
```ini
ExecStart=/opt/talent-graph/backend/.venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8200 --workers 1
MemoryMax=420M      # 内存上限，超出自动重启，保护小内存服务器
CPUQuota=120%
Restart=on-failure
```

### 5. 初始化数据（首次部署，建议本地执行后写入云 MySQL）

```bash
cd backend
uv run python data/generate_dataset.py      # 生成 JD 数据集
uv run python data/run_pipeline.py --reset   # 构建图谱
uv run python data/seed_relations.py         # 注入技能关系
```

## 三、方式二：Docker（容器化部署）

提供多阶段 `deploy/Dockerfile`（第一阶段构建前端，第二阶段运行后端并托管静态资源）与 `deploy/docker-compose.yml`（限制 420MB 内存 / 1 CPU）。

```bash
cd talent-graph-system
docker compose -f deploy/docker-compose.yml up -d --build
# 访问 http://<server-ip>:8200/
```

> 注意：在内存 < 2GB 的服务器上构建镜像可能紧张，建议本地构建镜像后 `docker save`/`docker load` 到服务器，或直接采用方式一。

## 四、配置项（backend/.env）

```ini
DB_HOST=...        DB_PORT=...     DB_USER=...   DB_PASSWORD=...   DB_NAME=talent_graph
DEEPSEEK_API_KEY=...    DEEPSEEK_BASE_URL=https://api.deepseek.com   DEEPSEEK_MODEL=deepseek-chat
TAVILY_API_KEY=...      SERPER_API_KEY=...
EMBED_API_KEY=...       EMBED_BASE_URL=http://.../v1   EMBED_MODEL=bge-small-zh-v1.5   EMBED_DIM=512
APP_PORT=8200
```

---

## 五、运维

```bash
systemctl restart talent-graph      # 重启
journalctl -u talent-graph -n 100   # 查看日志
systemctl show talent-graph -p MemoryCurrent   # 查看内存占用
```

**更新发布**：本地重新 `npm run build` → 替换服务器 `static/` 与 `app/` → `systemctl restart talent-graph`。
