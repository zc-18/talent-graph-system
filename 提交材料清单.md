## 五、一键复现

```bash
# 后端
cd backend && uv sync
uv run python data/generate_dataset.py        # 生成数据集
uv run python data/run_pipeline.py --reset      # 构建图谱（清洗→解析→交叉验证→落库）
uv run python data/seed_relations.py            # 注入技能关系
uv run python data/evaluate.py all              # 三项核心指标评测
uv run pytest --cov=app                         # 单元测试 + 覆盖率
# 前端
cd frontend && npm install && npm run build
```
